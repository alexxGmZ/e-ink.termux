# e-ink.termux

e-ink colorscheme for termux.

&nbsp;

## Installation

1. Clone or download your preferred colorscheme (light or dark).
2. Place the selected `color.properties` file in the `~/.termux` directory.
3. Restart termux.

&nbsp;
